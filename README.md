# Boxing Interval Timer – Website

Simple website for the Boxing Interval Timer app: main page, Privacy Policy, and Contact.

## Before you publish

1. **Set your contact email**  
   Edit `contact.html` and replace `support@example.com` with your real support or contact email (in both the visible text and the `mailto:` link).

2. **Icon**  
   Ensure `iconlight.png` is in this folder so it appears on the main page. If missing, copy it from your app’s `assets/iconlight.png`.

## Enabling GitHub Pages

1. **Commit and push**  
   Commit all files and push to GitHub (e.g. with GitHub Desktop).

2. **Turn on GitHub Pages**  
   - On GitHub.com, open this repository (**Boxing-Interval-Timer**).  
   - Go to **Settings** → **Pages**.  
   - Under **Source**, choose **Deploy from a branch**.  
   - Branch: **main** (or **master**), folder: **/ (root)**.  
   - Click **Save**.

3. **Your site URL**  
   After a minute or two, the site will be at:
   - `https://YOUR_USERNAME.github.io/Boxing-Interval-Timer/`

Use this URL in your App Store/Play Store listing for “Privacy Policy” and “Support” / “Contact” links.
